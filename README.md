# Digital-Literacy-Project
📌 Project Overview
This project was completed as part of the Digital Literacy course at VIT Bhopal University. The objective was to build a Digital Literacy Portfolio while acting as a Student Digital Ambassador.  

The project demonstrates essential digital skills, including content creation, professional online presence, coding practice, communication etiquette, and cyber safety awareness. All tasks are organised systematically in this repository.

🧩 Module-wise Summary
🔹 Task 1: Digital Literacy Infographic
Created a one-page infographic using Canva, highlighting:
- Meaning of digital literacy  
- Essential tools for students  
- Safe internet practices  
- Email etiquette  

📁 Folder: `task-1-presentation/

🔹 Task 2: Digital Portfolio Setup
Developed profiles on key platforms:
- GitHub (with profile README)  
- LinkedIn (education details updated)  
- Kaggle (account created)  

📁 Folder: `task-2-portfolio/`

🔹 Task 3: Platforms Exploration
- Completed a beginner coding challenge on HackerRank  
- Created a Google Form quiz on digital literacy  

📁 Folder: `task-3-platforms/`

🔹 Task 4: Email & Etiquette
- Drafted two professional emails (extension request & internship inquiry)  
- Created a social media Do’s & Don’ts checklist  

📁 Folder: `task-4-email-etiquette/`

🔹 Task 5: Cybercrime Awareness
- Prepared a case study on cybercrime (UPI fraud)  
- Created a prevention checklist with safety measures  

📁 Folder: `task-5-cybercrime/`
